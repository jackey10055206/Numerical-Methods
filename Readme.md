## 說明

* 使用方法
    * 直接執行main.py即可

* 使用六種方法分別去分析以下4條方程式
    * $e^x - 3x cos2x = 8.3$
    * $e^{xsinx} - xcos2x = 2.8$
    * $e^{sinx} + e^{cosx}$
    * $3sinx^2 + exp(cos(x^2))$
    
* 六種方法分別是
    * Bisection Method
    * FalsePosition Method
    * ModifyFalsePosition Method
    * Secant Method
    * Newton Method
    * FixedPoint Method
    
## 分析

* 依照電腦程式跑出來的結果，前5種方法都大同小異
* 精確度最高的是Bisection Method
* FixedPoint Method一直爆掉 生出來的答案都是無窮
